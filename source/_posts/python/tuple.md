## 一、最基础定义
元组 tuple = 不可变的有序序列

```python
t = (1, 2, 3)
t = 1, 2, 3   # 省略括号也可以
```
**核心特点**
- 不可变（不能修改、删除、添加元素）
- 有序
- 可重复
- 可存任意类型（int、str、list、dict、tuple...）
- 支持索引、切片、遍历
- 比列表更快、更安全

## 二、创建元组的 5 种方式
```python
# 1. 标准写法
t = (1, 2, 3)

# 2. 省略括号（最常用）
t = 1, 2, 3

# 3. 空元组
t = ()

# 4. 单元素元组（必须加逗号！！！）
t = (1,)   # 正确
t = (1)    # 错误！这是 int 不是元组

# 5. 构造函数
t = tuple([1,2,3])
```

## 三、元组支持的操作
t = (10,20,30)
1. 索引
2. 切片
3. 遍历:for i in t:
4. len(t)、max(t)、min(t)、sum(t)
5. 查找
    ```python
    20 in t   # True
    t.index(20)  # 找下标
    t.count(20)  # 统计次数
    ```

## 四、元组里的列表可以改
```python
t = (1, [2,3])
t[1].append(4)   # 可以！因为列表可变
print(t)  # (1, [2,3,4])
```
